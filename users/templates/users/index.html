Czym są Fajne Dane?

W serwisie tym udostępnione zostały zbiory danych oraz raporty przygotowane przez Sieć Obywatelską Watchdog Polska.

Dokumentacja API Przykłady użycia

Co znajdziemy w seriwisie?

W bazie danych serwisu Fajne Dane znajdziemy między innymi wyniki kampanii przeprowadzonych w ramach projektu Sprawdzamy Jak Jest, a także inne zbiory danych zebranę przez Sieć Obywatelska.

  • Zbiory danych zawierające, zweryfikikowane oznaczenia tysięcy dokumentów.
  • Zbiory danych dostępne w wygodnym formacie plików CSV.
  • Surowe dane, zawierające również odrzucone odpowiedzi.
  • Surowe dane, które można pobrać za pośrednictwem REST API.

Zasady stosowania API

W celu ograniczenia potęcjalnych nadużyć, wprowadzone zostały pewne zasady ograniczające stosowanie serwisu:

  • Dostęp jest darmowy, ale wymaga rejestracji
  • Ograniczona jest liczba zapytań do serwisu:
    • Użytkownik niezalogowany: 200 zapytań / dziennie,
    • Użytkownik zalogowany: 200 zapytań / godzine.
  • Dostęp do danych w postaci plików CSV nie jest ograniczony.
